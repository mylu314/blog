## <span id="jump2">Java variable<span>
  
  ### instance variable
  > 归属于实例化的对象  
  > 实例化时分配储存空间
 
  ### static variable
  > 归属于类，所有实例化的对象共用   
  > 类装载时分配储存空间

  ### block variable
  > 归属于局部的模块，比如方法内部

  ```java
  // code block to be executed
}
  ```

